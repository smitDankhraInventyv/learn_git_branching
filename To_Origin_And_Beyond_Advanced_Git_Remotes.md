# level 2 - To Origin And Beyond -- Advanced Git Remotes!

## 1: Push Main!

### steps
```
$ git checkout main

$ git pull

Fast forwarding...

$ git checkout side1

$ git rebase main

$ git checkout side2

$ git rebase side1

$ git checkout side3

$ git rebase side2

$ git checkout main

$ git rebase side3

Fast forwarding...

$ git push
```
